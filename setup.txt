--------------------------------------------------------------
Install XAMPP:
https://www.apachefriends.org/download.html

Goto local phpmyadmin:
http://localhost/phpmyadmin (This is where the data can be seen)
    -Create new database called "cardhub"
    
In XAMPP:
    - Click Explorer in the XAMPP control panel
    - Goto C:\xampp\htdocs
    - Create new folder called cardhub
    - copy database files (all .php) into it. Also copy the images folder into it.
    - Type this url once into your browser: http://localhost/cardhub/database.php?action=setup


FOR THE FIRST PAGE, OPEN THIS LINK: http://localhost/cardhub/start.html



Paste the following in phpmyadmin php section, this is readymade card data:
